HTML Text Formatting Tags

Below is a list of tags used to format text (display special types of text) in HTML

TAG NAME ACTION
b tag Bold Text Defines bold text, without extra importance
strong tag Important Text Defines a text with strong importance, it is displayed in bold
i tag Italic Text Defines text in an alternate voice or mood, it is displayed in italic
em tag Emphasized Text Defines text with an emphasis, it is also displayed in italic
small tag Small Text Defines small text, it displays text in smaller font
mark tag Mark Text Defines a text that should be marked or highlighted
del tag Deleted Text Defines text that has been deleted, it is displayed with a line striked through the text
ins tag Inserted Text Defines text that has been inserted, it is displayed with a line underneath the text
sub tag Subscript Text Defines a subscipt, it displays the text below the normal line and as a smaller font
sup tag Superscript Defines a superscript, it displays the text above the normal line and as a smaller font
